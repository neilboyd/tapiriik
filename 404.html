This site was an attempt to run a clone of tapiriik because that has stopped working reliably.

I cloned it, converted it to run on Kubernetes, added a service to sync to MapMyFitness, and had it running on this site.

But I realised that the code base is very old, most of the services are old and don't support their API's any more, and I don't like Python very much, and so it wasn't worth me spending money to run this service.

If you're looking for an alternative, I would suggest FitnessSyncer.
